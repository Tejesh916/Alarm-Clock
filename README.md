# Alarm Clock Web Application

This project is a simple and interactive **Alarm Clock Web Application** developed using **HTML, CSS, and JavaScript**. It allows users to set alarms with different notification sounds and provides a clean, user-friendly interface.

The application applies **fundamental Data Structures and Algorithms (DSA)** concepts for efficient time handling, alarm scheduling, and event management. The project focuses on organizing logic effectively while maintaining smooth user interaction.
